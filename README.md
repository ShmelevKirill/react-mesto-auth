# Проект: Mesto

Mesto - это интерактивная страница, созданная на React, где пользователи могут делиться фотографиями. В данной работе реализован функционал регистрации и авторизации;

## Технологический стек:
* HTML5, 
* CSS3, 
* Flexbox, 
* Grid Layout,
* Adaptive layout,
* React,
* JavaScript, 
* Webpack, 
* BEM,
* api;
